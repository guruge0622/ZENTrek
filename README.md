# ZENTrek

Simple static/PHP site for a travel/tour booking service.

## Overview
This repository contains the ZENTrek website files (HTML, CSS, PHP) intended to be run on a local PHP server such as XAMPP/Apache with PHP enabled.

## Quick start
- Place the project in your web server document root (e.g., `C:\xampp\htdocs\zentrek`).
- Start Apache and MySQL in XAMPP control panel if using dynamic features.
- Open http://localhost/zentrek/Home.html in your browser.

## Notes
- Some pages use PHP (e.g., `signup.php`, `login.php`, `processbooking.php`) and may require a configured database.
- The `ZENTrek_remote` folder is a local clone of an empty remote and is ignored via `.gitignore`.
